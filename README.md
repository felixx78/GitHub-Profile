## Tech Stack

- React
- TailwindCSS
- React Query
- React Router
- React Loading Skeleton


## Run Locally

Clone the project

```bash
  git clone https://github.com/v1st78/GitHub-Profile
```

Go to the project directory

```bash
  cd GitHub-Profile
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
